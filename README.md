# Housing-Resale-Price-Analysis-Singapore-
This project focuses on analyzing and comparing housing resale prices in Singapore. Given that property purchases are significant financial decisions, our goal is to help users better understand pricing patterns and be more prepared to pay for the amenities and features they value.

We conduct three main analyses:

Internal Factors Analysis

Examines how property-specific characteristics (e.g., size, layout, condition) influence resale prices.

External Factors Analysis

Investigates the impact of nearby amenities and infrastructure, such as supermarkets, transportation, and other points of interest.

Combined Analysis

Integrates both internal and external factors to identify the key drivers affecting housing resale prices.

Through these analyses, we aim to provide insights into which features have the greatest influence on property value in Singapore.
